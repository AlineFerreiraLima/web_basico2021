Protocolo HTTP e Servidores WEB (HTTP)

O que é e como funcionam os protocolos HTTP e HTTPS? O que faz o HTTP na estrutura cliente servidor? Como funcionam as sessões no HTTP? O que é um cookie? O que são servidores web e por que utilizar?

O que é e como funcionam os protocolos HTTP e HTTPS?

HTTP é um protocolo de transferência de hipertexto, sendo implementado nos software responsáveis por essa comunicação (navegadores e servidores web), a transferência HTTP não é segura, pois todas as informações transmitidas não são criptografadas, com isso você corre o risco de invasores tenham acesso a suas informações.

HTTPS também é um protocolo de transferência de hipertexto, porém sua utilização é segura pois possui o certificado SSL, no qual se trata de um fornecedor terceirizado para proteger uma conexão e verificar se o site é legítimo.

Funcionamento: Ambos funcionam por meio de solicitações, a solicitação é enviada para o servidor, e como resposta o servidor envia uma nova solicitação exibindo o resultado da sua pesquisa em forma de texto e imagens.

O que faz o HTTP na estrutura cliente servidor?

Realiza a troca de dados entre cliente-servidor, nesse contexto podemos exemplificar como: eu como cliente entro no meu navegador web e faço uma pesquisa, seja um endereço de um site ou a receita de bolo de chocolate, o HTTP então irá realizar por meio das solicitações o pedido por dados armazenados em um servidor, que por sua vez irá fazer a busca desses dados solicitados e por meio de uma nova solicitação enviará ao cliente a resposta de sua pesquisa, que no caso seria a receita de bolo de chocolate.

Como funcionam as sessões no HTTP?

As sessões são específicas de cada usuário, a mesma é capaz de armazenar informações do usuário conforme ele vai interagindo com a página, porém essas informações não são salvas de fato, apenas utilizadas no período de interação do usuário com a aplicação, após o usuário finalizar seu acesso as informações são descartadas.

O que é um cookie?

Um cookie é um arquivo de texto simples composto de acordo com as informações do endereço web visitado, ele pode armazenar informações básicas, como endereços IP e preferências de idiomas.

Essas informações são obtidas quando você acessa um site pela primeira vez, esse site então envia para o seu computador, celular e etc, um cookie com suas preferências e quando você voltar a acessar esse site o arquivo é lido e as suas informações são carregadas, liberando seu acesso mais rapidamente.

O que são servidores web e por que utilizar?

Os servidores web (HTTP) são softwares que processam pedidos via HTTP, os servidores web prestam serviços de armazenamento, processamento e entrega de páginas web aos clientes no qual é utilizado o protocolo HTTP como comunicação entre servidor e cliente, para isso é necessário uma conexão entre o computador onde está instalado o servidor web e o computador do cliente, com isso os servidores web precisam estar disponíveis dia e noite. A comunicação realizada pelo HTTP segue um modelo de request e response (pedido e resposta), essa conexão não é persistente, a cada acesso ou atualização do conteúdo é criada uma nova conexão HTTP e com isso todo o ciclo se repete.
